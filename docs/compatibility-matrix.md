# Compatibility Matrix

## Runtime and tooling

| Component | Supported / tested |
| --- | --- |
| Node.js | 22.x (CI baseline) |
| Phaser | ^4.0.0-rc.7 |
| Vite | >=5.0.0 |
| TypeScript | 6.x |

## Notes

- Vite is an optional peer dependency and required only for build-time SVG compilation plugin usage.
- Browser compatibility follows Phaser 4 compatibility.
- Update this table whenever peer dependency ranges change.
